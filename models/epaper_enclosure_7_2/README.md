# 7.2 寸墨水屏外壳

这是鲜知贴 7.2 寸墨水屏使用的两件式 3D 打印外壳，原始文件名为“冰箱显示屏”。
模型已经过多版实物打印和装配调整。

![7.2 寸墨水屏外壳实物](preview.png)

模型也发布在 MakerWorld：
[7.2 寸墨水屏外壳](https://makerworld.com.cn/zh/models/2726320-7-2-cun-mo-shui-ping-wai-ke#profileId-3162552)

## 文件说明

- `冰箱显示屏.step`：CAD 装配文件，包含“显示屏”和“后壳”两个实体。适合用
  Fusion 360 打开，检查结构、测量尺寸或继续修改。
- `冰箱显示屏.3mf`：可直接在 Bambu Studio 中打开的打印工程，包含显示屏前框
  和后壳两个对象。
- `preview.png`：从 3MF 工程内提取的实物预览图。

STEP 文件实测包络约为 `178.8 × 119.7 × 16.0 mm`。这里的包络尺寸是整个装配
的最大外形，不等于墨水屏玻璃尺寸或可视区域尺寸。

## 打印参考

当前 3MF 保存的主要打印参数为：

- 打印机：Bambu Lab P2S。
- 喷嘴：0.4 mm。
- 层高：0.16 mm，首层 0.20 mm。
- 墙层：2 圈。
- 填充：15%。
- 支撑：关闭。

不同材料、喷嘴、热床和机器状态会改变收缩与配合效果。建议先在 Bambu Studio
中重新检查摆放、切片预览和悬空区域，再发送打印；接口开孔、屏幕槽和前后壳配合
最好通过一次试装确认。

## 修改模型

需要修改结构时，优先在 Fusion 360 中打开 STEP。调整屏幕相关尺寸时，应把
“容纳整块屏幕的安装槽”和“正面显示内容的可视窗口”作为两组独立尺寸处理，
不要只用一个宽高参数同时控制两者。

如果需要重新设计不同尺寸的外壳，也可以参考
[4.2 寸模型的大模型建模方法](../ink_frame_v1/README.md#设计方法全程由大模型完成建模)，
测量屏幕、驱动板、电池、ESP32、接口及紧固件后，把结构和打印约束交给安装了
CAD Skill 的 coding agent 多轮生成与调整。
